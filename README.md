# Alef

Alef Component for Modern Web Apps, it's inspired by **React** and **Svelte**, base on **TSX**.

- Born in Typescript
- With Standard JSX Syntax
- AOT Compile in Rust
- No Virtual DOM
- Zero Runtime
- Reactive
- Support SSR
